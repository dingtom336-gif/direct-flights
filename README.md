# direct-flights

Search for non-stop, direct flights only — no layovers, no connections. For travelers who value time over cost.

## Overview

Search for non-stop, direct flights only — no layovers, no connections. For travelers who value time over cost. Wraps the flyai-cli to provide real-time travel data with booking links.

## Quick Start

### Install Skill

```bash
clawhub install direct-flights
# or
npx skills add alibaba-flyai/flyai-skill/tree/main/skills/direct-flights
```

### Install CLI

```bash
npm i -g @fly-ai/flyai-cli
```

### Verify

```bash
flyai --version
```

## What It Does

Search for non-stop, direct flights only — no layovers, no connections. For travelers who value time over cost.
